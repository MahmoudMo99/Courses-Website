Programming Concept

- Programming is the process of writing instructions and directing commands to a computer or any other device such as DVD readers
or audio and video receivers in modern communication systems, to direct this device and inform it how to deal with data or how to perform a series of required actions called an algorithm .

A brief history of programming

- Programmable devices have been around since at least 1206 AD .
- However, the first computer program dates back to 1843 .
- In the 1880s, Hermann Hollerith invented the concept of storing data in a machine-readable form .
- Machine code was the language of early software and was written in the instruction set of a given machine, often in binary encoding .
- Soon, assembly languages were developed that allow the programmer to specify instructions in text format .
- High-level programming languages have made the program development process simpler and more understandable .
- The Fortran programming language was the first widely used high-level language for practical implementation in 1957 .

Programming Languages
